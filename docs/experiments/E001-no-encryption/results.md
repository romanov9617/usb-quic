## Наблюдения
- Захвачена TCP-сессия между сервером и клиентом на порту 3240
- Отсутствуют признаки TLS/DTLS/QUIC
- Wireshark автоматически декодирует трафик как USB/IP

## Декодирование протокола

Wireshark успешно интерпретирует иерархию:
TCP → USBIP → USB → USB Request Block (URB)
Наблюдаются:
- `URB Submit`
- `URB Response`
- USB topology (bus / device / endpoint)

## USB Mass Storage
В payload присутствуют сигнатуры протокола USB Mass Storage Bulk-Only Transport:
- `USBC` (Command Block Wrapper)
- блочные операции чтения данных

## Интерпретация
Декодирование до уровня USB-команд без применения ключей или расшифровки подтверждает, что:
- данные передаются в открытом виде;
- USB/IP не обеспечивает конфиденциальность трафика.

## Итог
Гипотеза подтверждена: USB/IP не использует шифрование и допускает анализ USB-трафика стандартными средствами сетевого мониторинга.
