# USB/IP: подтверждение отсутствия шифрования

## Цель
Экспериментально подтвердить, что протокол USB/IP не использует шифрование и передаёт данные USB-устройств в открытом виде, доступном для анализа стандартными средствами сетевого мониторинга.

## Гипотеза
Если USB/IP не применяет криптографическую защиту, то сетевой трафик между сервером и клиентом может быть:
- захвачен на одном из легитимных участников соединения;
- декодирован до уровня USB/IP и USB Request Block (URB);
- проанализирован без применения ключей или расшифровки.

## Конфигурация стенда
- **USB/IP сервер**: локальный ноутбук (CachyOS Linux)
- **USB/IP клиент**: виртуальная машина Ubuntu 24.04 (VirtualBox)
- **USB-устройство**: USB-флешка (USB Mass Storage)
- **Сеть**: Bridged Adapter (через `wlan0`)
- **Транспорт**: TCP, порт 3240

## Вывод
Захваченный сетевой трафик был успешно декодирован в Wireshark до уровня `USBIP → USB → URB`, включая команды USB Mass Storage. Отсутствие TLS/DTLS/QUIC и возможность анализа полезной нагрузки подтверждают отсутствие шифрования в USB/IP.

## Связь с USB-over-QUIC
Эксперимент демонстрирует фундаментальный недостаток USB/IP — отсутствие конфиденциальности. USB-over-QUIC предполагается как решение данной проблемы за счёт использования QUIC с TLS 1.3 и сквозного шифрования.
